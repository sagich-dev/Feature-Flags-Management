# Features

Feature modules for the application. Each feature should have its own directory with components, hooks, and API logic.

## Structure

```
features/
└── [feature-name]/
    ├── components/
    ├── hooks/
    ├── api/
    └── index.ts
```

## Examples

- `feature-flags/` - Feature flag management
- `user-settings/` - User preferences and settings
- `notifications/` - Notification system
